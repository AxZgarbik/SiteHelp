Segmentación Environment


Clave de orientación Descripción Recomendación
Posición del anuncio Describe en qué parte de la página se encuentra el espacio publicitario: los valores los define el editor y no siempre se informan de manera confiable. Utilizar como segmentación INCLUDE o EXCLUDE
Ads.txt Utilizar como segmentación INCLUDE o EXCLUDE
Tipo de entorno El tipo de entorno es "In-App" o "Web". "In-App" es la mejor manera de orientar anuncios al inventario de aplicaciones móviles. "Web" hace referencia tanto a la Web de escritorio como a la Web móvil. Si desea orientar sus anuncios solo a la Web para computadoras de escritorio o dispositivos móviles, la mejor práctica es la orientación por capas en plataformas adicionales . Utilizar como segmentación INCLUDE o EXCLUDE
Interstitial Si la ubicación es una ubicación interstitial o no. Los interstitial son comunes en los entornos de aplicaciones móviles, pero también se pueden utilizar en entornos web móviles y de escritorio. Utilizar como segmentación INCLUDE o EXCLUDE
Tipo de interstitial El espacio interstitial acepta banner, video o ambos Utilizar como segmentación INCLUDE o EXCLUDE
Native Layout

Sólo disponible en Line Item Native.

Nota: Este tipo de orientación solo está disponible para ciertos intercambios y no se transfiere a todas las solicitudes de oferta nativas, por lo que su uso puede limitar el tráfico. 		Utilizar como segmentación INCLUDE o EXCLUDE
Rewarded Placement Reward Utilizar como segmentación INCLUDE o EXCLUDE
Top frame

Describe el inventario donde el espacio publicitario NO está en un iFrame. La segmentación por fotograma superior puede resultar útil en los casos en que la creatividad no interactúa bien en entornos iFrame.

Nota: excluir el marco superior es equivalente a apuntar a iFrame.

 Utilizar como segmentación INCLUDE o EXCLUDE
Api Framework

Los marcos de API que están permitidos para la ubicación. Los valores incluyen MRAID 1, MRAID 2, MRAID 3, OMID (también conocido como SDK de medición abierta), ORMMA, Desconocido, VPAID 1 y VPAID 2. El campo es más útil para asegurarse de que está entregando en ubicaciones que permiten que ciertos scripts se ejecuten, como la medición de la viewbility .

 Utilizar como segmentación INCLUDE o EXCLUDE

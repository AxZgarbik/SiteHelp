Segmentación


Segmentación Geografica


Adsmovil aumenta la subasta con datos geográficos de un proveedor de datos de terceros (MaxMind), por lo que los datos geográficos casi siempre están presentes para la orientación.

Clave de orientación Descripción Recomendación
País País del usuario Utilizar como segmentación INCLUDE o EXCLUDE
Region / Estado Región del usuario. En los Estados Unidos, esta es la abreviatura de estado de dos letras. Utilizar como segmentación INCLUDE o EXCLUDE
Metro Código de metro del usuario. Basado en los códigos metropolitanos de Google que son similares a los DMA de Nielsen. Solo disponible en el mercado estadounidense. Utilizar como segmentación INCLUDE o EXCLUDE
Cuidad Cuidad del usuario Utilizar como segmentación INCLUDE o EXCLUDE
Lista de latitud y longitud Lista de coordenadas de latitud y longitud. Utilizar como segmentación INCLUDE o EXCLUDE
Latitud y longitud presentes Presencia de valores de latitud y longitud Utilizar como segmentación INCLUDE o EXCLUDE
Tipo de ubicación Tipo de ubicación, por ejemplo, servicios de ubicación GPS, dirección IP o datos proporcionados por el usuario Utilizar como segmentación INCLUDE o EXCLUDE
Código postal Código postal o código postal del usuario. Los códigos postales se pueden ingresar directamente en el campo "Código postal" como una lista separada por comas. La segmentación por código postal también se adapta a los códigos postales internacionales, pero comuníquese primero con el servicio de asistencia de Adsmovil para obtener información específica sobre los formatos de valores coincidentes para varios países. Utilizar como segmentación INCLUDE o EXCLUDE
Lista de códigos postales

Para cualquier conjunto de códigos postales superiores a 100, recomendamos crear una "Lista de códigos postales". Para crear una lista:

  1. Haga clic en el botón + Nuevo y seleccione "Lista" en el menú desplegable
  2. Seleccione "Lista de códigos postales" en el menú desplegable Tipo de lista
  3. Dé un nombre a su lista y haga clic en "Guardar y continuar"
  4. Agregue sus códigos postales en el campo "Agregar elemento de lista" (uno por fila) O cargue un archivo .txt de cremalleras (también uno por fila) O cargue un archivo .csv de cremalleras (también uno por fila)

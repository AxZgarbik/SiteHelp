Segmentación Plataforma


La segmentación por plataforma se utiliza para garantizar que se oriente al dispositivo o plataforma deseados, incluidos los tipos de dispositivos y los sistemas operativos.

Esta función amplía las capacidades de orientación geográfica de Adsmovil para ir más allá de las geografías estándar (es decir, estado, ciudad, código postal, dma, etc.) y le permite definir sus propios radios geográficos para la orientación. Esta función es nativa de la plataforma y no conlleva ningún cargo de CPM adicional.

Clave de orientación Descripción Recomendación
Banda ancha Ancho de banda del dispositivo, por ejemplo, red celular, Ethernet o WiFi Utilizar como segmentación INCLUDE o EXCLUDE
Navegador Navegador web del usuario (Chrome, Safari) Utilizar como segmentación INCLUDE o EXCLUDE
Versión del navegador Versión del navegador del usuario Utilizar como segmentación INCLUDE o EXCLUDE
Portador Operador de dispositivo o ISP, p. Ej. VERIZON Utilizar como segmentación INCLUDE o EXCLUDE
Tipo de dispositivo El tipo de dispositivo desde el que se originó la impresión, por ejemplo, teléfono móvil, CTV, tableta, etc. Utilizar como segmentación INCLUDE o EXCLUDE
Marca del dispositivo Marca del dispositivo, por ejemplo, Apple. Utilizar como segmentación INCLUDE o EXCLUDE
Modelo de dispositivo Modelo del dispositivo, p. Ej. IPhone. Utilizar como segmentación INCLUDE o EXCLUDE
Tamaño de pantalla del dispositivo

Dimensiones físicas del tamaño de la pantalla. Los tamaños son (en el área de píxeles):

XL - Mayor que 640001
L - De 250001 a 640000
M - De 90001 a 250000
S - Menor o igual a 90000
Utilizar como segmentación INCLUDE o EXCLUDE
Sistema operativo Especifica el sistema operativo del dispositivo. Utilizar como segmentación INCLUDE o EXCLUDE
Versión del sistema operativo Especifica la versión del sistema operativo del dispositivo (iOS 10, Android Honeycomb, etc.) Utilizar como segmentación INCLUDE o EXCLUDE

Prácticas recomendadas para la segmentación por plataforma

Hay varias formas de orientar anuncios a aplicaciones móviles, web móvil o inventario web de escritorio. A continuación, se muestran las mejores prácticas que optimizan tanto el alcance como la precisión:

Aplicación móvil

Para el inventario de aplicaciones móviles, la mejor práctica es utilizar la clave de segmentación "Tipo de entorno" dentro del módulo de segmentación "Inventario".

Específicamente, seleccione el valor "En la aplicación" de la clave de orientación "Tipo de entorno".

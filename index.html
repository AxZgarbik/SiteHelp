Configuracion de Usuario


Como crear un nuevo usuario

Si es un usuario con privilegios de administrador o tiene derechos de creación de usuarios, podrá crear nuevos usuarios para la interfaz de usuario y la API del DSP Adsmovil.

Para crear un usuario:

  1. Vaya a Admin -> Usuarios desde la barra de navegación

  2. Esto lo llevará a la página de Usuarios . Haga clic en + Nuevo -> Usuario para abrir el cuadro de diálogo 'Crear nuevo usuario'.

  3. Complete todos los campos relevantes (campos obligatorios marcados con un asterisco rojo):

    1. Primer nombre
    2. Apellido
    3. Email
    4. Role
      1. El rol de un usuario es una plantilla separada que gobierna qué acciones pueden o no pueden tomar. Para obtener más información sobre los roles, consulte aquí .
    5. Identificación alternativa
      1. La identificación alternativa se puede establecer en un valor de su elección, o se puede dejar en blanco si no es relevante.
      2. Puede ser útil para integraciones personalizadas, como si utiliza Adsmovil junto con su propia interfaz de usuario patentada (que tiene ID de usuario existentes)
    6. Notas
      1. Cualquier nota adicional que desee dejar para un usuario (por ejemplo, que haga referencia a su organización / nivel de acceso, etc.)
    7. Multicuenta
      1. Esto determina si el usuario tiene acceso a varias cuentas en Adsmovil, si se ha creado más de una.
    8. Activo
      1. Determina si el usuario está configurado como activo (solo superusuarios). Para los usuarios normales, esta no será una opción, y el nuevo usuario deberá verificar su correo electrónico a través de un token de inicio de sesión que activará automáticamente su usuario.

  4. Envíe la nueva configuración de usuario.

Luego, se enviará un correo electrónico a la dirección de correo electrónico que proporcionó en el campo Correo electrónico, y se le pedirá al destinatario que verifique su correo electrónico y defina una contraseña para usar con la plataforma. Una vez que hayan hecho esto, su cuenta se activará y podrán iniciar sesión en la plataforma con su combinación de correo electrónico y contraseña.

Segmentación


Segmentación Latitud y longitud


La segmentación por radio geográfico le permite definir sus propias áreas geográficas para fines de segmentación. Por ejemplo, es posible que desee orientar sus anuncios a cualquier usuario que se encuentre a menos de 0,5 kilómetros de la ubicación de una tienda determinada con un cupón móvil.

Esta función amplía las capacidades de orientación geográfica de Adsmovil para ir más allá de las geografías estándar (es decir, estado, ciudad, código postal, dma, etc.) y le permite definir sus propios radios geográficos para la orientación. Esta función es nativa de la plataforma y no conlleva ningún cargo de CPM adicional.

  1. En el nivel de lista, establezca el radio predeterminado para cada elemento de la lista. Cada elemento de la lista individual puede tener su propio radio personalizado. Cualquier elemento de lista sin un radio especificado heredará el radio predeterminado establecido en el nivel de lista

    • Los usuarios de radio mínimo pueden establecer: 0.001 kilómetros
    • Radio máximo que los usuarios pueden establecer: 50 kilómetros

  2. Agregue Elementos de la lista a su lista. Los elementos de la lista se pueden agregar de varias maneras:

    • Escriba la dirección en el campo Buscar ubicación. Una vez que encuentre la ubicación, haga clic en Agregar elemento de lista para agregar la ubicación a su lista. Si se muestran varios "pines" en el mapa (por ejemplo, si busca "NYC Subway Station"), puede hacer clic en el pin que le interesa y luego hacer clic en Agregar elemento de lista

    • Al hacer clic en Agregar elemento de lista sin escribir una dirección en el campo Buscar ubicación, obtendrá un elemento de lista vacío donde puede especificar latitud, longitud, radio, valor y nombre del elemento de lista directamente en el formulario. Como cualquier otro elemento de la lista, el valor se puede utilizar como modificador de oferta. Tenga en cuenta que el campo de nombre de elemento de lista también está disponible en todos los tipos de elementos de lista y se puede utilizar como un identificador para su elemento de lista (p. Biblioteca Pública de Nueva York).


    • Cargue un archivo csv, .txt, .xls o .xlsx. El orden de las columnas es el mismo que aparece en la interfaz de usuario (latitud, longitud, radio, valor, nombre del elemento de la lista). Cada tipo de archivo requiere un formato ligeramente diferente, cuyos detalles puede encontrar aquí .

    La plataforma valida que no existen elementos de lista duplicados exactos dentro de una sola lista. Un elemento de lista de latitud / longitud duplicado solo se produciría si la latitud, la longitud y el radio son exactamente iguales, incluidos los puntos decimales.

  3. Editar elementos de la lista
    1. Una vez que se guarda un elemento de la lista, solo puede editar el valor o el nombre del elemento de la lista. Si necesita editar la latitud, la longitud o el radio, debe eliminar el elemento de la lista y crear uno nuevo.
    2. Al hacer clic en el icono del mapa se muestra una pequeña vista previa del radio geográfico
  4. Oriente la lista personalizada en el módulo de segmentación geográfica. Tenga en cuenta que si se incluyen varias listas Lat / Long, O las listas se combinarán.
